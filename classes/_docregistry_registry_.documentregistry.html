listWidgetFactories

  • listWidgetFactories(ext?: string): string[]
    • Defined in docregistry/registry.ts:224

    List the names of the registered widget factories.

    Parameters

    • Default value ext: string = "*"

      An optional file extension to filter the results.

    Returns string[]

    A new array of registered widget factory names.

    Notes

    The first item in the list is considered the default. The returned list has widget factories in the following order:

    • extension-specific default factory
    • global default factory
    • all other extension-specific factories
    • all other global factories
